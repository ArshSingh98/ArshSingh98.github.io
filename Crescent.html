Crescent God View

Developed frontend for the web application used as Managerial Application for Tabletop RPG utilizing Vue.js, Bootstrap, & JavaScript.

Software & Algorithm

As a Game Master for Crescent, it is difficult to manage the stats and current conditions of all of the game's players. Currently, in order to properly manage a game, one must have several different tabs open and scroll through multiple pages to find needed information. Additionally, it is difficult for a Game Master to to add or edit information on players' character sheets since they would need all of the players' login info to do so.

The app contains an easy-to-use user platform with a responsive UI, allowing the Game Master easy navigation to players' character sheets and helpful, real-time updates on players' conditions.
